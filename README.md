 ## CLI 명령어
  * mider(디렉토리 생성)
  * cd(디렉토리 이동)
  * rmdir(디렉토리 삭제)
  * touch(파일 생성과 내용입력)
  * cp(파일 복사)
  * mv(파일 이동 및 이름변경)
 ## GIT의 활용
  **git init -저장소 초기화**  
  **git status -상태 확인**   
  **git add -인텍스로 이동**    
  **git commit -m -커밋 및 메세지로 입력**   
  **git push -원격 저장소로 전송**

## HTML 기본 문법  
 ++<!DOCTYPE html>
<html lang="ko-KR">

<head>
    <meta charset="UTF-8">
    <title>기본구조</title>
    <link rel="stylesheet" href="css/style.css">
</head>

<body>
    <h1>커피</h1>
    <p>커피는 마시면 기운이 솟아나요..</p>
</body>

</html>
'''
